<h2>1. Общо описание</h2>
<p>Целта на проекта е да бъде създадена конзолна игра, подобна на TBS (Turn-based strategy) игрите. В играта ще се изправят една срещу друга две армии, като битките се развиват на ходове.</p>
<br>
<h2>2. Описание на задачата</h2>
<p>Трябва да бъде реализирана следната функционалност: </p>
